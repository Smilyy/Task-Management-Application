Django Task Manager

This is a Django-based web application for task managing.

  • 1. Create, view, and filter tasks by category.
  • 2. Only admins can manage categories.
  • 3. Bootstrap styled UI.
  • 4. Login required for admin page.

Quick Start


pip install pipenv
pipenv install django 
pipenv shell
python3 manage.py migrate
python3 manage.py createsuperuser
python3 manage.py runserver

Security

Admin users are automatically logged out when they leave the admin panel for security. CSRF protection and custom error pages are also implemented.

Test This App

Use test.http (included in the 'task_management/' folder) with VS Code REST Client to test all project URLs.
